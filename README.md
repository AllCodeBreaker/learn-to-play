# College Management System 

It is an advanced CMS which not only manage the college students details and performance 
also manage and track the skills development of the students. It has a gamified technical challenges 
and points with strike which make them engaged in the challenges. 



## Tech Stack 
### Front-end 

- React + vite
- Tailwind css
- Material UI
- GSAP


### Backend 
- Flask
- Sqlalchemy 
- MongoDB
- Redis


### Other tools
- Docker
- Github Actions


## Highlights 

- Gamified Learning experience
- Guide how to solve real problems
- Subject based projects and skills
- Job preparation and guide
- Tracking both academic and professional performance
